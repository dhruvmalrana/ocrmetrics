Evaluation Method

This tool uses standard OCR evaluation methodology with exact word matching:

  • Words are preprocessed (case normalization, punctuation removal based on config)
  • Only exact matches count toward Precision and Recall
  • Exact matches = 0 character errors (words are identical)
  • Unmatched words = all characters count as errors

Metrics Explained

Precision What % of OCR words are correct?
Precision = Exact Matches / Total OCR Words
Example: OCR outputs "hello world test", GT has "hello world" ‚Üí 2/3 = 66.67%
Recall What % of GT words were found?
Recall = Exact Matches / Total GT Words
Example: GT has "hello world test", OCR outputs "hello world" ‚Üí 2/3 = 66.67%
F1 Score Balanced measure of precision and recall
F1 = 2 √ó (Precision √ó Recall) / (Precision + Recall)
Single metric for comparing OCR models
CRR (Character Recognition Rate) Character-level accuracy
CRR = 1 - (Character Errors / Total GT Characters)
Exact matches have 0 errors (identical words)
Character errors = 0 for exact matches + all chars in unmatched GT words + all chars in unmatched OCR words

Complete Example

Ground Truth: "The quick brown fox"

OCR Output: "The quik brown"

Word Matching:

  • Exact matches: "The", "brown" (2 words match exactly)
  • Unmatched in GT: "quick", "fox"
  • Unmatched in OCR: "quik"

Metrics:

  • Precision: 2/3 = 66.67%
  • Recall: 2/4 = 50.00%
  • F1 Score: 57.14%
  • CRR:
    ‚Ä¢ Matched: "The"‚Üî"The" (0 errors), "brown"‚Üî"brown" (0 errors)
    ‚Ä¢ Unmatched GT: "quick" (5 chars), "fox" (3 chars) = 8 errors
    ‚Ä¢ Unmatched OCR: "quik" (4 chars) = 4 errors
    ‚Ä¢ Total: 12 errors / 16 GT chars = 25% CRR

Quick Reference

  • High Precision: OCR doesn't add extra/wrong words
  • High Recall: OCR doesn't miss words
  • High F1: Good balance of both
  • High CRR: Few character-level errors
  • Color coding: Normal = exact match, Red = no match
